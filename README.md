# ☁️ Explorador AWS Cloud Practitioner & AI Study Companion

Un panel interactivo (Single Page Application) diseñado para transformar la preparación del examen **AWS Certified Cloud Practitioner**. 

Este proyecto no es solo un resumen de conceptos; es una herramienta diseñada para integrarse en un ecosistema de aprendizaje moderno donde **la Inteligencia Artificial actúa como tu tutor personal**.

## 🤖 El paradigma del aprendizaje con IA

La memorización de manuales técnicos extensos ha quedado atrás. Este panel ha sido estructurado para ofrecer la información tal y como la procesaría un sistema de aprendizaje optimizado:
* **Definiciones precisas:** Lo estrictamente necesario.
* **Analogías del mundo real:** Para generar anclajes mentales fuertes y rápidos.
* **Datos Clave (Snippets):** La información exacta que los algoritmos de evaluación de AWS suelen utilizar en sus preguntas trampa.

## 💎 Integración con Gemas de Gemini

Para cerrar el ciclo de aprendizaje (Teoría -> Práctica -> Validación), este proyecto se apoya en el uso de **Gemas de Gemini** (versiones personalizadas de la IA de Google). 

En lugar de buscar exámenes de prueba estáticos en internet, puedes utilizar una IA configurada específicamente como tu "Examinador AWS".

### ¿Cómo utilizar el ecosistema completo?
1. **Estudia la teoría** interactuando con los conceptos de este panel (`index.html`).
2. **Abre la Gema de Evaluación:** 👉 [Gema: Preparación AWS Cloud Practitioner](https://gemini.google.com/gem/1sLtTCycklsQqW6N1ZKLiMx2S3efmpNR2?usp=sharing)
3. **Ponte a prueba:** Pídele a la Gema que te haga un test de 10 preguntas sobre un bloque específico (por ejemplo, *Seguridad*). La IA no solo te dirá si has acertado, sino que te explicará el razonamiento arquitectónico detrás de cada respuesta fallida.

## 🚀 Cómo desplegar este proyecto

Este proyecto es totalmente "Serverless" (sin servidor) y no requiere instalación de dependencias.

* **Uso local:** Descarga el archivo `index.html` y ábrelo en cualquier navegador web.
* **Uso web (Live Demo):** Puedes acceder a la versión desplegada gratuitamente mediante GitHub Pages aquí: `[INSERTA_AQUÍ_TU_ENLACE_DE_GITHUB_PAGES]`
